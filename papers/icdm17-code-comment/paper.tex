	\begin{abstract}
In this paper, we propose a framework to automatically generate descriptive
comments for source code blocks. While this problem has been studied by
by many researchers previously, their methods are exclusively based on 
fixed template. Our framework, which doesn't rely on any template, makes use
of a new recursive neural network called Code-RNN to extract features
from the source code and embed them into one vector. When the result of
Code-RNN is connected to a basic recurrent neural network (RNN), the overall
framework generates text descriptions of the code with accuracy significantly
higher than baseline methods using sequence-to-sequence RNN. The Code-RNN
model can also be used in other scenario where the representation of code
is required.
\end{abstract}
